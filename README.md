# netwrap workspace

Monorepo for the `netwrap` package and test apps.

## Structure

- `packages/netwrap`: the library package
- `apps/web`: Next.js test app
- `apps/api`: Node.js test app
- `apps/mobile`: React Native CLI test app

## Quick start

```bash
npm install
```

```bash
npm run build -w netwrap
```

## App commands

```bash
npm run dev -w web
```

```bash
npm run start -w netwrap-api
```

```bash
npm run start -w MobileApp
```
